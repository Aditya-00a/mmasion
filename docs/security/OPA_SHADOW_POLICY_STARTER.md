# OPA Shadow Policy Starter

Status: Future scope

This document is the planned home for baseline OPA-style shadow policies that can validate MMASION decisions in parallel.

Planned rule categories:

- unsupported-claim blocking
- uploaded-file scope enforcement
- intervention escalation thresholds
- human-checkpoint requirements
- sensitive-data handling
- policy override audit requirements
- model/provider fail-open prevention

Goal:

Compare MMASION runtime decisions against explicit policy rules before enterprise deployment.
